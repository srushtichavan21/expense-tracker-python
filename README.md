# 📊 Expense Tracker (Python)

A simple Python-based application to track daily expenses and manage spending efficiently.

---

## 🚀 Features

* Add daily expenses
* View all expenses
* Categorize spending
* Calculate total expenses

---

## 🛠 Tech Used

* Python
* JSON (file handling)

---

## ▶️ How to Run

```bash
python main.py
```

---

## 💡 Example

Food - ₹200 (Lunch)
Travel - ₹100 (Bus)

Total Spending: ₹300
